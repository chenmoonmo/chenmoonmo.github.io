2020-11-30

  • CSS隐藏scrollbar

  • git 常用操作汇总

    • git checkout -b 'name' 新建并切到name分支
    • git branch -d 'name' 删除name分支
    • git fetch 拉取远程分支
    • git merge 'name' 合并当前分支与name分支
    • git diff 'name' 比较分支
    • git pull 拉取远程分支并合并 相当于git fetch + git merge
    • git reset HEAD～1 --hard 回退操作 push除外
    • git reset --hard [commitId] 回退到指定版本

  • location.reload()

    强制浏览器刷新

  • Vue 父子传值 Props是单向数据流，如果要改变Props的值

    • 使用$emit传递事件，让父组件改变
    • 使用computed改变或者使用prop属性初始化新的data
